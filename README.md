Fortnite (DMA) is an experimental project focused on **DMA-based tooling and external system interaction**.
This repository contains documentation and example components related to working with **DMA devices and secondary processing systems**.

---

## 📌 Overview

This project explores concepts such as:

* ⚙️ External system communication
* 💻 DMA hardware workflows
* 🖥️ Multi-PC processing setups
* 🧩 Software architecture for external tools

The goal of the project is to experiment with **low-level system interaction and hardware-assisted workflows**.

---

## ✨ Features

### 🎯 Aimbot

* Triggerbot: Auto-fires when your crosshair is over an enemy.

### 👁️ Visuals

* Player ESP: 2D Box, Corner Box, Snaplines, and full Skeleton bones.
* Info Display: Distance, Weapon Name, Rank, Level, Kill score, TeamID and Platform.
* Color-Coding: Different colors for Visible, Hidden, and Knocked players.
* Fov Arrows: Directional indicators for enemies outside your screen.
* Loot ESP: Filtered by rarity (Common to Mythic) and Category (Ammo, Heals, Weapons, Chests, Supply Drops, Llamas).
* Spectators list

### 📡 Radar

* 2D Map: A full 300x300 desktop radar overlay.
* Positioning: Move the radar anywhere on your screen.
* Enemy Direction: See exactly which direction enemies are facing.

### ⚡ Exploits

* Experimental "Shoot Through Walls" capability via memory-write.

## 🛠️ Setup

### 1️⃣ Download the Files

* Download the repository files.
* Extract them to a working directory on your **secondary system**.

### 2️⃣ Launch the Program

* Ensure all files remain in the same directory.
* Run the executable to start the software.

⚠️ **Important Notice**

Do **not** launch the executable without the required **DLL files** included.
If files are missing, the software may fail to start.
